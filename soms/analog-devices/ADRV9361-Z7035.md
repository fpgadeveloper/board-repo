## Extras
- Marvell 88E1512 10/100/1000 Mbps Ethernet PHY (RGMII to Zynq PS GEM, magnetics + RJ45 on carrier)
- Microchip USB3320 USB 2.0 ULPI PHY (Zynq PS USB0, OTG, USB receptacle on carrier)
- 4x Zynq GTX gigabit serial transceiver lanes (up to 6.6 Gbps) brought to micro-headers; 2x MGT reference clock inputs
- AD9361 auxiliary 12-bit ADC and 2x 10-bit auxiliary DACs brought to micro-headers
- 4x AD9361 GPO pins (2.5V default, up to 3.3V) brought to micro-headers
- Zynq XADC differential analog input (V_0_P/N) brought to micro-headers
- 8x U.FL miniature coaxial RF connectors (4x TX, 4x RX) plus 2x TX monitor inputs; 10MHz to 6GHz via Mini-Circuits TCM1-63AX+ baluns
- 40 MHz on-board crystal as AD9361 reference clock (external clock 10-80 MHz also supported via JX header)
- 33.333 MHz PS clock, 25 MHz Ethernet clock, 24 MHz USB clock oscillators
- Boot mode select via on-SoM switches S3/S4 (Cascade JTAG / QSPI / SD); SD-source select switch S1 (SoM µSD vs carrier SD)
- PS_POR_B push-button (SW1) for manual Zynq hard reset; FPGA_DONE status LED (D4)
- 4x FCI 0.8mm BergStak 100-pin micro-header receptacles (JX1-JX4) exposing 193 PL SelectIO + 12 PS MIO + 4 AD9361 GPO pins (209 user I/O total)
- 20-layer PCB, 62mm × 100mm (Dangerous Prototypes "Sick of Beige" DP10062 compatible)

## Notes

The Zynq XC7Z035-2L FBG676I is a -2 speed grade, low-voltage binned device with industrial temperature grade. The AMD canonical ordering format is `XC7Z035-2LFBG676I` (the Analog Devices wiki spells this `-L2` in places but the orderable part is `-2L`).

This SoM is designed to mate with a carrier card via four 100-pin micro-header receptacles. Compatible Analog Devices carriers include the ADRV1CRR-FMC (full-featured FMC carrier), ADRV1CRR-BOB (breakout carrier), PZSDRCC-PCIE (PCIe carrier), and PZSDRCC-PackRF (PackRF carrier). The SoM is **not** pin-compatible with standard PicoZed (non-SDR) carrier cards.

The RF signal chain (8x U.FL connectors for TX/RX) operates 70 MHz–6 GHz with channel bandwidths from <200 kHz to 56 MHz.
