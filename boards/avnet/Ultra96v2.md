## Notes

### Configuration

Boot mode is determined by DIP switch labelled SW2.

| Boot mode          | 1 | 2 |
|--------------------|---|---|
| JTAG               | 1 | 1 |
| SD Card            | 0 | 1 |
| USB                | 0 | 0 |

Notes:

* 0 = OFF
* 1 = ON
