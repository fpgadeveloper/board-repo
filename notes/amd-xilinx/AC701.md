## Configuration

FPGA configuration is determined by DIP switch SW1.

| Config mode        | 1 | 2 | 3 |
|--------------------| --| --| --|
| JTAG               | 1 | X | 1 |
| Master SPI         | 0 | X | 1 |

Notes:

1. Switch 2 is not used
2. X = don't care
3. 0 = OFF position
4. 1 = ON position
