## Configuration

Boot mode is determined by DIP switch SW11.

| SW11               | 1 | 2 | 3 | 4 | 5 |
|--------------------|---| --| --| --| --|
| JTAG  (default)    | 0 | 0 | 0 | 0 | 0 |
| JTAG (independent) | 1 | 0 | 0 | 0 | 0 |
| QSPI               | 0 | 0 | 0 | 1 | 0 |
| SD                 | 0 | 0 | 1 | 1 | 0 |
